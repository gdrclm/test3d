# READ FIRST

Этот пакет задаёт систему, по которой Codex должен строить браузерные 3D-модели из чертежных видов.

## Главная цель
Не просто получить одноразовый mesh, а получать **структурированные procedural-модели**, которые потом можно:
- перекрашивать
- шейдерно переоформлять
- деформировать
- анимировать
- переиспользовать в других сценах
- перестраивать по новым чертежам без переписывания всей системы

## Жёсткие принципы
1. **Чертеж важнее красоты.** Сначала соблюдается форма и пропорции.
2. **Шаблон важнее импровизации.** Codex обязан выбирать template family.
3. **Модель обязана быть редактируемой.** Нельзя отдавать только голый финальный mesh без recipe/part graph.
4. **Материалы и шейдеры отделены от геометрии.**
5. **Анимация и деформация закладываются заранее.**
6. **Один объект = один source recipe + один runtime builder.**
7. **Все размеры и pivots нормализованы.**
8. **Модель должна быть пригодна для браузерного продакшена.**

# TEMPLATE LIBRARY

Ниже описаны базовые семейства шаблонов, из которых должны строиться почти все объекты.

## templateBed
Состав:
- frame
- mattress
- blanket
- pillow slots
- optional plush slot

Подходит для:
- кровать
- детская кровать
- диван-кровать

## templateCabinet
Состав:
- body
- top
- door panels / drawer panels
- handle system
- optional foot/base

Подходит для:
- шкаф
- тумбочка
- комод
- ящик для игрушек

## templateDesk
Состав:
- top
- leg/support system
- optional drawer block
- desktop sockets

Подходит для:
- письменный стол
- рабочий стол
- маленький столик

## templateBookshelf
Состав:
- outer frame
- shelves
- optional back panel
- shelf sockets

Подходит для:
- книжный шкаф
- открытый стеллаж

## templateRoundTable
Состав:
- top
- central support or legs
- optional rim

## templateSoftSeat
Состав:
- base soft mass
- secondary support mass
- seam / compression zones

Подходит для:
- пуфик
- кресло-мешок
- мягкая подушка-сиденье

## templateFlatWallObject
Состав:
- slab/frame
- mount offset
- surface plane

Подходит для:
- постер
- часы
- холст
- окно как плоский объект

## templateCurtain
Состав:
- rail reference
- vertical cloth volume
- wave profile
- optional tie section

## templateLamp
Состав:
- base
- stand
- shade
- optional bulb/glow slot

## templateSmallProp
Состав:
- main mass
- accent mass
- optional micro detail

Подходит для:
- игрушка
- кубик
- чайник
- ваза
- коробка
- тапочки

## Правила выбора шаблона
1. Сначала по функции объекта.
2. Потом по доминирующей массе.
3. Потом по профилю спереди/сбоку.
4. Потом по footprint сверху.

Codex не должен создавать новые template families без веской причины.

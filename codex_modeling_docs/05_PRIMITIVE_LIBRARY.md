# PRIMITIVE LIBRARY

## Разрешённые базовые примитивы

### roundedBox
Главный примитив для стилизованной мебели.

Параметры:
- width
- height
- depth
- radius
- segments

### softBox
Похож на roundedBox, но мягче и пухлее. Для текстиля и мягких объектов.

### panelRect
Тонкая панель:
- фасад ящика
- экран
- постер
- дверца

### cylinder
Для опор, ножек, стоек, абажуров.

### softCylinder
Для мягких или стилизованных цилиндрических форм.

### sphere / halfSphere
Для округлых игрушек, ламп, декора.

### capsule
Для мягких продолговатых форм.

### cone
Для абажуров, пирамидок, декоративных форм.

### frameRect
Рамка с внутренней пустотой.

### blob
Свободная мягкая форма. Использовать ограниченно.

### latheProfile
Для вращательных тел:
- ваза
- бутылка
- простая лампа

## Допустимые shape ops

- squash
- stretch
- symmetry mirror
- safe merge

## Запреты
1. Не использовать хаотичные boolean-операции без необходимости.
2. Не делать слишком высокий polycount ради мелкой детали.
3. Не пытаться моделить всё одним огромным мешем.
4. Не запекать стиль в геометрию, если эффект можно оставить на материале/шейдере.

## Общие стилистические правила
- 1 крупная масса
- 1–3 средние массы
- немного мелких акцентов
- силуэт важнее микродетали
- читаемость сверху и в изометрии важнее фронтальной красоты

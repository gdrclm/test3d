# OBJECT CATALOG TEMPLATE

Использовать этот шаблон на каждый из 51 предмета.

## Карточка объекта

### ID
`obj_XX_name`

### Название
Краткое имя предмета.

### Family
Например:
- furniture
- decor
- lighting
- toy
- textile
- wallObject

### Template
Например:
- templateCabinet
- templateDesk
- templateSoftSeat
- templateSmallProp

### Обязательные размеры
- width
- height
- depth

### Required parts
Список логических частей.

### Required sockets
Список точек крепления и размещения.

### Material slots
Какие material slots обязан поддерживать объект.

### Deform zones
Какие зоны мягкие, жёсткие, полумягкие.

### Animation hooks
Если есть.

### Poly budget
Целевой и максимальный.

### Notes
Любые ограничения по интерпретации.
